# Kishan Patel — Dark Professional One‑Page Site

This folder contains a dark, modern one-page website with hero image, about, services, and contact.

## Customize
- Replace `assets/portrait.jpg` with a real photo (same filename).
- Edit text/links in `index.html`.

## Publish
- **Vercel (free):** https://vercel.com → New Project → drag this folder → Deploy.
- **GitHub Pages (free):** upload files → Settings → Pages → Branch `main`, folder `/`.
- **Wix (embed):** Editor → Add → Embed Code → Embed a Site → paste your deployed URL.
